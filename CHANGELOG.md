# Changelog

## 1.1.0

- **Feature (Server Control)**: Added manual Start/Stop/Restart commands for the VS Code server.
- **Feature (WSL Support)**: Added ability to configure the Server Host (IP address) in the Chrome extension options.
- **UI/UX**: Updated Status Bar to show "UI SmartPicker" and added toggle functionality via click.
- **Performance**: Improved structural search by excluding common build folders (`.next`, `dist`, `out`, etc.).

## 1.0.0

- Initial release of **UI SmartPicker**.
- Feature: Structural UI element picking and matching.
- Feature: Automatic AI prompt generation with screenshot and context.
- Support: Chrome and VS Code extension synchronization.
