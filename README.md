# 📘 UI SmartPicker: AI 기반 UI 편집 도구 (VSCode & Chrome)

**UI SmartPicker**는 브라우저와 VS Code를 연결하여 개발 효율을 극대화하는 도구입니다. 브라우저에서 수정하고 싶은 UI 요소를 클릭하면, 해당 위치의 소스 코드를 자동으로 찾아주고 AI에게 전달할 완벽한 프롬프트를 생성해 줍니다.

## ✨ 주요 특징

- **정확한 코드 탐색**: HTML 구조와 텍스트 분석을 통해 클래스명이 다르거나 복잡한 JSX 구조에서도 정확한 소스 코드 위치를 찾아냅니다.
- **스마트 캡처**: 요소 주변의 맥락이 포함되도록 상하좌우 여백을 조절하여 스크린샷을 생성합니다.
- **간결한 AI 프롬프트**: 파일 위치(@파일명:줄번호), URL, 이미지 경로를 포함한 최적화된 프롬프트를 자동으로 클립보드에 복사합니다.
- **다크 테마 UI**: 세련된 토스트 알림과 직관적인 피드백을 제공합니다.

---

## 🛠️ 구성 요소

### 1. UI SmartPicker (VSCode Extension)
- 로컬 서버를 실행하여 브라우저로부터 데이터를 수신합니다.
- 프로젝트 내 소스 코드를 검색하고 해당 줄로 즉시 이동합니다.
- AI 프롬프트를 생성하여 클립보드에 복사합니다.

### 2. UI SmartPicker (Chrome Extension)
- 웹페이지에서 마우스 클릭으로 UI 요소를 선택합니다.
- 선택된 요소의 정보를 캡처하여 VS Code로 전송합니다.

## 🚀 시작하기

### 1. 설치 및 실행 (개발자 모드)

#### VS Code 확장 프로그램
1. 프로젝트를 VS Code에서 엽니다.
2. `npm install`로 의존성을 설치합니다.
3. `F5` 키를 눌러 [Extension Development Host] 창을 실행합니다.

#### Chrome 확장 프로그램
1. Chrome 브라우저에서 `chrome://extensions`에 접속합니다.
2. '개발자 모드'를 활성화합니다.
3. '압축해제된 확장 프로그램을 로드'를 클릭하고 `ui-smart-picker/chrome-extension` 폴더를 선택합니다.

### 2. 사용 방법
1. VS Code 상태바에 **`Picker: 54321`** 아이콘이 켜져 있는지 확인합니다.
2. 브라우저에서 확장 프로그램 아이콘을 눌러 활성화합니다 (배지가 초록색으로 변함).
3. 수정하고 싶은 웹페이지의 UI 요소를 클릭합니다.
4. VS Code 알림과 함께 클립보드에 프롬프트가 복사됩니다. AI에게 붙여넣고 수정을 요청하세요!

## ⚙️ 설정

- **포트 번호**: 기본값은 `54321`입니다. 충돌이 발생할 경우 VS Code 설정(`uiSmartPicker.port`)과 Chrome 확장 프로그램 옵션에서 변경할 수 있습니다.
